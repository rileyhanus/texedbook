Abstract

This is not a full thesis template! It only demonstrates how to create per-chapter references using the chapterbib package with BibTeX. (Do not use with BibLaTeX!)

Each chapter must be in its own .tex file and include-d into the main .tex file. If compiling on your own machine, run bibtex on each generated .aux file, before running pdflatex twice more. (These are done automatically on Overleaf.)
